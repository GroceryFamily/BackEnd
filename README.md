## Secrets

Create `secrets/application-secrets.yaml` in the project's root directory:

```yaml
spring.datasource:
  username: MYSQL_USERNAME
  password: MYSQL_PASSWORD
```

Replace `MYSQL_USERNAME` and `MYSQL_PASSWORD` with actual values.
